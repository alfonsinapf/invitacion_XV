¿Por qué ocurre este error?

El error 403 (Forbidden) indica que no tienes permisos para realizar la acción solicitada. En tu caso:

1 La playlist pertenece a una cuenta familiar de Spotify, no a la cuenta del desarrollador registrada en Spotify Developer Dashboard.
2 Spotify tiene restricciones específicas para playlists en cuentas familiares para evitar modificaciones no autorizadas.
3 Falta de autorización explícita del propietario de la playlist para tu aplicación.

Alternativa de Playlist

Crear una nueva playlist en la cuenta del desarrollador como alternativa.

Código de la solución implementada:

// Función mejorada para manejar errores de permisos async function addToPlaylist(trackUri, trackName, artistName, durationMs, button) { try { // Primero verificamos los permisos const hasPermission = await checkPlaylistPermissions(); if (!hasPermission) { // Si no tenemos permisos, mostramos el formulario alternativo showAlternativeOption(trackName, artistName); return; } // Si tenemos permisos, procedemos con la adición const response = await fetch( `https://api.spotify.com/v1/playlists/${SPOTIFY_CONFIG.PLAYLIST_ID}/tracks`, { method: 'POST', headers: { 'Authorization': `Bearer ${spotifyAccessToken}`, 'Content-Type': 'application/json' }, body: JSON.stringify({ uris: [trackUri], position: 0 }) } ); if (response.status === 403) { throw new Error("No tienes permisos para modificar esta playlist. " + "El propietario debe autorizar la aplicación."); } if (!response.ok) { const errorData = await response.json(); throw new Error(errorData.error?.message || "Error al añadir la canción"); } // Éxito showSuccessMessage("¡Canción añadida exitosamente!"); } catch (error) { console.error("Error al añadir la canción:", error); showErrorMessage(`Error: ${error.message}`); } }

Próximos Pasos

Para resolver completamente el problema, sigue estas instrucciones:

1 Pide al propietario de la playlist que autorice explícitamente tu aplicación.
2 Haz que la playlist sea colaborativa en la configuración de Spotify.
3 Como alternativa, crea una nueva playlist en la cuenta del desarrollador.
